# Examples

Use these patterns as a starting point for agents that need to pay many services during a task.

## Buyer Responsibilities

- Maintain a funded collateral position on the networks you want to use.
- Register a 4Mica client scheme for every network your target sellers accept.
- Sign only payment requirements that match your agent policy.
- Track open tabs and settlement obligations before withdrawing collateral.

## TypeScript Example

```ts
const response = await fetchWithPayment("https://api.example.com/data");
if (!response.ok) {
  throw new Error(`Paid request failed: ${response.status}`);
}
const data = await response.json();
```

## Operational Notes

Use small prices and testnet collateral first. Once the flow is stable, add spend limits per task, allowlists for seller domains, alerts for failed settlements, and a withdrawal process that checks open tabs before moving funds.
