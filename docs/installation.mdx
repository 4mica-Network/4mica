# Installation

Install 4Mica packages for each supported language. This root-level page is kept for existing links; the organized navigation points to the newer sectioned docs where applicable.

## Key Points

- TypeScript: pnpm install @4mica/x402 @x402/fetch @x402/axios viem
- Python: pip install "4mica-x402[fastapi]" "4mica-x402[requests]" sdk-4mica
- Rust: cargo add sdk-4mica alloy

## 4Mica Context

4Mica is the clearing house for the agentic economy: a credit layer for x402 that lets agents and applications transact instantly, earn yield on collateral, and settle net positions later.
