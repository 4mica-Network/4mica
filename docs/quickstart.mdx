## Prerequisites

Before integrating 4Mica, ensure you have:

- A 4Mica account with API credentials
- Node.js 18+, Python 3.8+, or Rust 1.70+
- A wallet with some testnet funds (for testing)
- Basic understanding of HTTP payment protocols

## 30-Second Setup

<Steps>
  <Step title="Install the SDK">
    Choose your language and install the 4Mica package:

    <CodeGroup>
      ```bash npm
      npm install @4mica/x402
      ```
      ```bash pip
      pip install 4mica-x402
      ```
      ```bash cargo
      cargo add sdk-4mica
      ```
    </CodeGroup>
  </Step>
  <Step title="Register Your Facilitator">
    Create a facilitator account to participate in the 4Mica network:

    ```typescript
    import { facilitatorClient } from '@4mica/x402';

    const facilitator = await facilitatorClient.register({
      name: 'My Service',
      webhookUrl: 'https://your-domain.com/webhooks',
    });
    ```
  </Step>
  <Step title="Protect Your First Route">
    Add payment protection to any endpoint:

    ```typescript
    import { paymentMiddleware } from '@4mica/x402';

    app.get('/api/protected', 
      paymentMiddleware(),
      (req, res) => res.json({ data: 'Protected content' })
    );
    ```
  </Step>
  <Step title="Test It">
    Make a request and capture the 402 response:

    ```bash
    curl -i https://your-domain.com/api/protected
    # Returns HTTP 402 with payment requirements
    ```
  </Step>
</Steps>
