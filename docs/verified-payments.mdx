# Verified Payments

Use validation-gated guarantees for work that requires quality checks. This root-level page is kept for existing links; the organized navigation points to the newer sectioned docs where applicable.

## Key Points

- Use V2 guarantees when payment should depend on validation.
- Bind validator, agent, score threshold, and job hash.
- Ensure validation status is posted before remuneration.
- Use this for paid work where delivery alone is not enough.
