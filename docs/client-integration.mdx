# Client Integration

Wrap HTTP clients so they can pay 4Mica-protected routes. This root-level page is kept for existing links; the organized navigation points to the newer sectioned docs where applicable.

## Key Points

- Register FourMicaEvmScheme for the network.
- Wrap fetch, Axios, requests, or httpx.
- Let the wrapper create unique request IDs, sign guarantees, and retry paid requests.
- Apply agent spending policy before signing.

## 4Mica Context

4Mica is the clearing house for the agentic economy: a credit layer for x402 that lets agents and applications transact instantly, earn yield on collateral, and settle net positions later.
