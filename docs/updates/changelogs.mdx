# Changelogs

Track product, SDK, protocol, and network changes that affect builders. Changelog entries should help users answer one question quickly: do I need to change anything?

## Latest Changes

No dated release entries have been added yet.

## Categories

- Protocol: contracts, guarantees, validation, and settlement windows.
- SDKs: TypeScript, Python, Rust, middleware, and client wrappers.
- Facilitator: supported endpoints, payloads, networks, and versions.
- Networks and assets: new deployments, tokens, or configuration changes.
- Security: changes that affect keys, signatures, claims, or access control.

## Entry Format

- Date and release name.
- What changed.
- Who is affected.
- Required migration steps.
- Links to examples, API changes, or release notes.
