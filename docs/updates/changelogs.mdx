  ## Meet the new 4Mica website

  We redesigned the 4Mica website to make it easier to understand, explore,
  and build for the agentic economy.

  This update includes:

  - Clearer navigation across the website and developer documentation.
  - Refreshed product, solution, careers, and company pages.
  - A new documentation experience powered by Mintlify.
  - Updated legal pages for our terms, privacy policy, data processing agreement,

  - Improved mobile navigation, page layouts, and accessibility.

  No migration is required. Explore the new website at
  [4mica.io](https://4mica.io) or continue with the
  [getting started guide](/getting-started/introduction).
