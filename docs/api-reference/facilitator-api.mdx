# Facilitator API

Resource servers use the facilitator to connect x402 HTTP payment flows with 4Mica credit.

## Base URL

Hosted deployments commonly use <code>https://x402.4mica.xyz/</code>. Self-hosted or environment-specific deployments expose the same endpoint shape.

## Endpoints

- GET /health: liveness check.
- GET /supported: supported scheme, network, and x402 version combinations.
- POST /verify: validate the decoded payment payload against payment requirements.
- POST /settle: settle a signed claim and return the certificate required for enforcement.
