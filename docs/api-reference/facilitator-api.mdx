# Facilitator API

Resource servers use the facilitator to connect x402 HTTP payment flows with 4Mica credit. Buyers usually reach it indirectly through the seller advertised tab endpoint.

## Base URL

Hosted deployments commonly use <code>https://x402.4mica.xyz/</code>. Self-hosted or environment-specific deployments expose the same endpoint shape.

## Endpoints

- GET /health: liveness check.
- GET /supported: supported scheme, network, and x402 version combinations.
- POST /tabs: open or reuse a payment tab for a payer, recipient, asset, and network.
- POST /verify: validate the decoded payment payload against payment requirements.
- POST /settle: settle a signed claim and return the certificate required for enforcement.

## Tab Request Shape

```json
{
  "network": "eip155:84532",
  "x402Version": 2,
  "userAddress": "0xUser",
  "recipientAddress": "0xRecipient",
  "erc20Token": null,
  "ttlSeconds": 86400
}
```
