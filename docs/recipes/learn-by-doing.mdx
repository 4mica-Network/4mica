# Learn by Doing

The fastest way to understand 4Mica is to build the payment loop yourself once: protect a route, pay it from a funded wallet, inspect the tab, and settle the result.

## Exercise 1: Protect A Mock Route

Create a route that returns premium JSON and protect it with a small Base Sepolia price. Confirm that an unpaid request returns HTTP 402.

## Exercise 2: Pay From A Test Wallet

Deposit test collateral, wrap your HTTP client, and call the protected route. Confirm that the client opens a tab and retries automatically.

## Exercise 3: Break One Thing

Try an unsupported network, missing collateral, or wrong recipient address. Good integrations are easy to debug when the payment flow fails.
