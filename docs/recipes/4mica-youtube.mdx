# 4Mica YouTube

Use videos to show the flow that text cannot always make obvious: an unpaid request becomes a 402, the buyer retries with a signed guarantee, and the seller serves the paid response.

## Suggested Playlists

- What 4Mica is and why agentic payments need credit.
- x402 basics: HTTP 402, payment requirements, and paid retries.
- Building a pay-per-use API with 4Mica middleware.
- Connecting an agent wallet and depositing collateral.
- Understanding guarantees, clearing cycles, settlement, and default coverage.

## What A Good Demo Shows

A good video should show both sides of the flow: the seller route configuration and the buyer request. It should end with the user understanding why 4Mica is faster than account setup, prepaid balances, or on-chain settlement per request.
