# Demo Apps

Demo apps should make the 4Mica story visible: agents can access useful resources immediately, sellers get guarantees, and settlement happens later.

## Paid API Gateway

A small gateway protects multiple upstream routes with different prices and logs each paid request by tab and request ID.

## Agent Data Buyer

An agent buys data from several APIs during one task, drawing from one collateral position instead of creating accounts with each provider.

## Settlement Dashboard

A dashboard shows open tabs, settled guarantees, paid tabs, remunerations, and withdrawal state for a seller or operator.
