## Overview

The Facilitator API provides endpoints for managing your payments, tabs, and settlements in the 4Mica network.

## Authentication

All requests require a Bearer token or facilitator secret:

```bash
Authorization: Bearer {facilitator_secret}
```

## Endpoints

### Get Facilitator Details

Retrieve information about your facilitator account.

```http
GET https://api.4mica.xyz/facilitators/{facilitatorId}
```

**Response:**
```json
{
  "id": "fac_xxx",
  "name": "My Service",
  "domain": "https://api.myservice.com",
  "webhookUrl": "https://api.myservice.com/webhooks/4mica",
  "status": "active",
  "schemes": ["sch_xxx"],
  "createdAt": "2024-01-01T00:00:00Z"
}
```

### List Payment Tabs

Get all active payment tabs for your facilitator.

```http
GET https://api.4mica.xyz/facilitators/{facilitatorId}/tabs
```

**Query Parameters:**
- `status` - Filter by status: `pending`, `funded`, `claimed`, `expired`
- `limit` - Number of results (default: 50)
- `offset` - Pagination offset

**Response:**
```json
{
  "data": [
    {
      "id": "tab_xxx",
      "amount": "0.01",
      "currency": "ETH",
      "status": "funded",
      "user": "0x...",
      "createdAt": "2024-01-01T00:00:00Z",
      "expiresAt": "2024-01-02T00:00:00Z"
    }
  ],
  "total": 150,
  "limit": 50,
  "offset": 0
}
```

### List Guarantees

Retrieve payment guarantees issued by your facilitator.

```http
GET https://api.4mica.xyz/facilitators/{facilitatorId}/guarantees
```

**Query Parameters:**
- `status` - Filter by status: `pending`, `verified`, `settled`, `failed`
- `scheme` - Filter by scheme ID
- `limit` - Number of results (default: 50)
- `offset` - Pagination offset

**Response:**
```json
{
  "data": [
    {
      "id": "gua_xxx",
      "tabId": "tab_xxx",
      "schemeId": "sch_xxx",
      "amount": "0.01",
      "currency": "ETH",
      "status": "verified",
      "signature": "0x...",
      "validationPolicy": {
        "method": "erc8004",
        "address": "0x..."
      },
      "createdAt": "2024-01-01T00:00:00Z",
      "verifiedAt": "2024-01-01T00:10:00Z"
    }
  ],
  "total": 523,
  "limit": 50,
  "offset": 0
}
```

### Settle Payments

Initiate payment settlement for guaranteed transactions.

```http
POST https://api.4mica.xyz/facilitators/{facilitatorId}/settle
```

**Request Body:**
```json
{
  "guaranteeIds": ["gua_xxx", "gua_yyy"],
  "batchId": "batch_xxx",
  "metadata": {
    "period": "2024-01-01",
    "reason": "daily_settlement"
  }
}
```

**Response:**
```json
{
  "settlementId": "set_xxx",
  "guaranteeIds": ["gua_xxx", "gua_yyy"],
  "totalAmount": "0.02",
  "currency": "ETH",
  "status": "processing",
  "estimatedCompletion": "2024-01-01T12:00:00Z"
}
```

### Create Withdrawal Request

Request to withdraw funds from your facilitator account.

```http
POST https://api.4mica.xyz/facilitators/{facilitatorId}/withdrawals
```

**Request Body:**
```json
{
  "amount": "0.5",
  "currency": "ETH",
  "recipientAddress": "0x...",
  "metadata": {
    "reason": "fund_sweep"
  }
}
```

**Response:**
```json
{
  "withdrawalId": "wth_xxx",
  "amount": "0.5",
  "currency": "ETH",
  "status": "pending",
  "transactionHash": null,
  "createdAt": "2024-01-01T00:00:00Z"
}
```

### List Webhooks

Retrieve all webhooks configured for your facilitator.

```http
GET https://api.4mica.xyz/facilitators/{facilitatorId}/webhooks
```

**Response:**
```json
{
  "data": [
    {
      "id": "wh_xxx",
      "url": "https://api.myservice.com/webhooks/4mica",
      "events": ["payment.verified", "tab.opened", "settlement.completed"],
      "status": "active",
      "lastDelivery": "2024-01-01T10:00:00Z"
    }
  ]
}
```

## Error Responses

All errors follow this format:

```json
{
  "error": {
    "code": "INVALID_REQUEST",
    "message": "The request was invalid",
    "details": {
      "field": "amount",
      "reason": "must be greater than 0"
    }
  }
}
```

### Common Error Codes

| Code | Status | Description |
|------|--------|-------------|
| `INVALID_REQUEST` | 400 | Request body or parameters are invalid |
| `UNAUTHORIZED` | 401 | Authentication failed or credentials missing |
| `FORBIDDEN` | 403 | You don't have permission for this resource |
| `NOT_FOUND` | 404 | The requested resource doesn't exist |
| `CONFLICT` | 409 | The request conflicts with current state |
| `RATE_LIMIT` | 429 | Too many requests |
| `SERVER_ERROR` | 500 | Internal server error |

## Rate Limiting

API requests are rate limited to:
- **Standard**: 100 requests per minute
- **Enterprise**: Contact sales for higher limits

Rate limit information is provided in response headers:
```
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1609459200
```

## Pagination

List endpoints support cursor-based pagination:

```http
GET https://api.4mica.xyz/facilitators/{id}/tabs?limit=50&offset=100
```
