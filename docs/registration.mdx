# Registration

Register a wallet by depositing collateral. This root-level page is kept for existing links; the organized navigation points to the newer sectioned docs where applicable.

## Key Points

- Choose a supported network.
- Connect or configure the payer wallet.
- Approve an <Tooltip headline="ERC-20" tip="Ethereum's standard interface for fungible tokens. It defines common functions for balances, transfers, approvals, and allowances so wallets and applications can interact with tokens consistently.">ERC-20</Tooltip> if needed.
- Deposit collateral into the Core4Mica flow.
- Wait for finalization before making paid requests.

## 4Mica Context

4Mica is the clearing house for the agentic economy: a credit layer for x402 that lets agents and applications transact instantly, earn yield on collateral, and settle net positions later.
